# VirtualRobotics (Autonomous Agent in Unity)

## Prerequisites
- uv tool [Installation](https://docs.astral.sh/uv/getting-started/installation/)
- Python 3.X (will be automatically downloaded with uv after running the project)

## Setup & Usage
1. Run the server: `uv run main.py`
2. Run the [Unity client](https://github.com/VirtualRobotics/VirtualRobotics-Unity)
